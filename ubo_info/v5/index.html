Développeur web
Developpeur Frontend
  • HTML5 & CSS3
  • Javascript (jQuery)
  • Plus loin
    • Bootstrap
    • TypeScript
    • ReactJS, AngularJS, VueJS
    • NodeJS (Pourquoi pas ?)
    • Webpack, GruntJS ou Gulp, NPM
Développeur Backend
  • PHP ou Python ou JAVA
  • SQL(MySQL, PostgreSQL)
  • NoSQL(MongoDB, Redis, ..) Optionnel
  • Plus loin
    • Symfony ou Django ou J2E
    • Apache
Développeur logiciel
  • Python
  • C ou C++ (Gestion de la mémoire: pointeurs et références)
  • JAVA
  • Analyse logicielle: UML
  • Bases de données: SQL ou NoSQL
  • Notions de réseaux

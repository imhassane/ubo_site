Mes débuts avec JAVA et UML

Écrit par Hassane SOW le 29 Novembre 2017 à 09:11

Il est temps de parler d’un de mes langages favoris qui est sûrement l’un des plus matures et immortels de la programmation, évidemment c’est le JAVA qui propose une infinité de possibilités à ces utilisateurs à travers des milliers de modules intégrées en son sein. Le JAVA je l’ai connu bien avant le Python ou le C++, il sert notamment à faire des logiciels portables sur PC ou sur mobile, il est même celui conseillé dans les développements android avec Kotlin le nouveau langage à la mode.

J’adore la syntaxe de ce langage et surtout la manière dont il gère les fichiers avec son module PATH, le JAVA est sûrement et sans contexte mon préféré pour le developpement de logiciels sous WINDOWS et sur PC en général, son modèle objet est fascinant et super facile à apprenhender et sa puissance est irrévocable, j’imagine que c’est pour cela certains le considereront comme langage immortel de nos jours et est presque tout le temps le langage le plus utilisé dans le monde avant d’être détrôné en 2017 par python.

Il est aussi utilisé dans le domaine du web et très apprecié par les entreprises professionnelles avec J2E qui est aussi simple à apprendre. En général, apprendre le JAVA est facile et pas compliqué, c’est un langage vraiment séduisant et puissant une fois qu’on l’a maitrisé. Sa communauté est immense et active à la fois tout probleme rencontré lors du developpement JAVA est susceptible d ‘être resolu sur des gorums comme stackoverflow et console. Sa syntaxe est similaire à celle de beaucoup d’autres langages qui se sont inspirés de lui comme le php et le C# qui le ressemble fortement par exemple et de ce fait quitter le java pour un autre langage facilite la compréhension de celui-ci.

Sans compter que JAVA couvre la majorité des concepts de la programmation excepté quelques uns considérés comme difficile à comprendre comme la notion d’héritage multiple et celle des pointeurs. Son système de garbage collector est d’une aide sans équivalent pour un développeur venant de C++ par exemple. En somme le langage JAVA est vraiment adapté pour tous les besoins et facile à apprendre.

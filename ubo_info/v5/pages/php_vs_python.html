PHP ou Python ? Mon expérience

Écrit par Hassane SOW le 29 Novembre 2017 à 13:52
Je vais maintenant faire une petite comparaison entre PHP et PYTHON, je tiens à dire que cette comparaison n’engage que mon avis personnel et tout le monde peut avoir un avis différent selon son expérience et je sais que l’avis de chacun est super important. J’ai commencé à coder à PHP à mes 14 ans et en PYTHON à 16 ans donc j’ai juste deux ans d’expérience dans chacun des langage mais croyez moi c’est vraiment suffisant.

Le temps d’apprentissage

Pour moi, c’est peut-être parce que je connaissais une panoplie de langages avant de l’apprendre mais le Python est largement plus facilement à apprendre que le PHP la plupart des programmeurs affirment que le Python est sûrement le langage de programmation le plus facile à apprendre et l’un des plus rapide. En PHP, il faut apprendre souvent plein de choses sans compter les erreurs qui sont rares par exemple en Python.

La communauté

En général, les deux communautés sont très grandes et actives et toutes les questions qui vous posent problèmes seront résolues, donc j’ai pas grand-chose à dire là-dessus.

Les frameworks

Niveu framework PHP est vraiment meilleur que Python, puisque PHP est spécialisé dans le WEB et que Python quant à lui n’est pas forcément spécialisé dans le web mais aussi dans les logiciels pc ou smartphones, l’automatisation des tâches et plusieurs autres tâches. Avec PHP, vous pouvez soit choisir CodeIgniter qui est super simple à apprendre et parfait pour commencer, Symfony un peu professionnel et beaucoup utilisé, CakePHP que j’ai d’ailleurs utilisé à mes débuts ou encore Laravel que j’ai jamais utilisé mais qui est quand-meme super bon ou encore pleins d’autres frameworks comme WordPress alors qu’en Python vous avez le choix entre Django qui est exellent et meilleur à mon goût que les frameworks PHP et Flask qui permet de créer un site en quelques minutes.

Développements en PHP VS en Python

Si PHP a beaucoup plus de frameworks que Python, vous devez alors savoir que developper une application Python est beaucoup plus simple en Python qu’en PHP, par sa simplicité et la lisibilité d'un code Python celui-ci est plus simple et beaucoup plus maintenable qu’un code équicalent en PHP et moins long d’ailleurs. Le developpement d ‘applications en Python est meilleur que celui en PHP car souvent un code PHP est moins lisible et son ordre dépend du developpeur sans compter que pour les debutants PHP est vraiment pas facile à apprendre comme langage.

Documentation

La documentation des deux langages est super bien faite et c’est très facile de se reperer car souvent des exemples sont fournis pour chaque notion expliquée. Donc de ce côté aucun soucis à se faire.

Conclusion

Vu le niveau que Python est entrain d’acquérir dans le monde de la programmation, il serait préferable pour toute personne voulant se lancer dans la programmaton de commencer par celui-ci d’autant plus qu’il est très faicile à apprendre et sa compréhension n’est que très facile, une grande communauté et super active est à votre service pour tout problème rencontré. Des sites openclassrooms, github et stackoverflow vous permettront d’avoir tout ce que vous voulez savoir.
